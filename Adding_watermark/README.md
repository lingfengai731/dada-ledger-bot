# DADA Watermark 🔏

给 DADA 的婚礼 / 花艺作品加水印的小工具。

最近我们的图在各个社交平台被人盗用,有的还直接说成是自己的作品。这个工具就为两件事:让盗图更费劲,以及万一真被盗,我们能拿出证据说"这是我们的"。

办法是给每张图叠两层水印:

- **看得见的那层**:角落一枚淡淡的 DĀDA ISLAND 白色 Logo(透明度/大小/位置可调)。挡不住有心人,但能劝退顺手转发的。
- **看不见的那层**:藏在像素里的一段编码,肉眼完全看不出来。图被压缩、转发之后它还在;把盗图拖进验证页就能解出来,证明出处。

每张图的隐形编码会记到一张本地登记表,对应"哪场婚礼、哪天"。所以验证时不只是告诉你"是 DADA 的",还能说出"这是 6 月 16 号 Komaneka 那场 Coco 的图"。

## 快速开始

```bash
pip install -r requirements.txt
python -m uvicorn app:app --port 8000
```

打开 http://localhost:8000,两个页面:

- **加水印**:选一张图,填上是哪场婚礼(不填也行),提交。几秒后下载成品,隐形编码已经登记好。
- **验证盗图**:把可疑的图拖进来。是我们的,它会显示编码 + 哪场婚礼;不是,它会直接说没找到水印。

想确认隐形水印真的扛得住压缩,又不想开浏览器,跑一下:

```bash
python test_roundtrip.py
```

它会加完水印,再模拟平台把图压成 q80、q60,看还能不能解回来。

## 它扛得住什么

隐形水印用的是频域算法(DWT-DCT),实测:

| 盗图常见的折腾 | 还认得出吗 |
|---|---|
| 重新压缩(转发、截图最常见) | 能,压到 q60 都行 |
| 稍微缩放 | 大多能 |
| 大幅裁剪、旋转、只截一小角 | 多半不行 |

说句实话:没有哪种隐形水印是抹不掉的。它真正解决的是最常见的那种盗图——整张图被截走、重新压缩、再发出去。要做到"裁了一角也认得出"、还能全网自动揪盗图,那得上 Digimarc、Imatag 这类收费服务,以后需要再说。

## 它是怎么搭的

- `watermark.py` —— 水印引擎。先用 Pillow 叠可见 Logo,再用 `blind-watermark` 把隐形编码埋进频域。
- `app.py` —— FastAPI 网页,三个接口:加水印、下载、验证,外加一张 SQLite 登记表。
- 验证用的是**模糊匹配**。压缩之后解出来的编码偶尔会差一两个字符,所以拿它跟登记表里每个编码比"差几个字",最接近、又足够近的就算命中。这样压过的图也认得出。

## 上线前记得两件事

1. `watermark.py` 里的 `PASSWORD_IMG`、`PASSWORD_WM` 是水印的钥匙。换成只有你们知道的值,别外传——谁拿到这两个数,谁就能读、甚至伪造水印。
2. 登记表 `registry.db` 和上传 / 输出目录都带着客户和作品信息,已经写进 `.gitignore`,不会进 git。
